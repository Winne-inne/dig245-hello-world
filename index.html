Tutorial Summary

What I've learned from the Keyboard shortcuts and Workflow tutorial is that workflow is necessary for successful coding. A good workflow not only speeds up the progress but ensures that time is not wasted trying to operate between the many files and windows programming inevitably brings. Instead of searching for the file icon, looking for the save button, or going across the screen to close a tab, there are commands on the keyboard that allow you to do all of that and more without having to move your hands too much, leaving your flow uninterrupted.It's good to set up a good workflow and familiarity with keyboard commands early in your coding endeavors to ensure success down the road.
